# ThermaStay – Spa & Hot Springs Finder

ThermaStay is a premium, minimalist luxury web application designed to help users find their perfect thermal escape. Built with a focus on high-end aesthetics, nature-inspired design, and seamless user experience, it offers a curated collection of wellness sanctuaries, specifically localized for **Navi Mumbai**.

---

## Project Architecture

The project is built using a clean, modern tech stack without external frameworks, ensuring maximum performance and full control over the aesthetic details.

### **Tech Stack**
- **HTML5**: Semantic structure for SEO and accessibility.
- **CSS3**: Custom design system featuring:
    - **Glassmorphism**: Elegant translucent navigation and overlays.
    - **Cinematic Animations**: Custom keyframes for steam effects and smooth transitions.
    - **Responsive Grid**: Fluid layouts for mobile, tablet, and ultra-wide screens.
- **Vanilla JavaScript**: Robust interactivity including:
    - **Advanced Filtering**: Real-time spa card filtering by temperature and accessibility.
    - **Booking System**: Interactive modal management for private tub reservations.
    - **Page Transitions**: Custom viewport wipes for seamless navigation.
    - **Intersection Observer**: Scroll-based reveal animations for premium "fade-in" effects.

---

## Key Features

### **1. Advanced Filter System**
Located on the `filters.html` page, this allow users to find spas based on:
- **Temperature Range**: 30°C to 50°C+.
- **Accessibility & Features**: Wheelchair access, senior-friendly amenities, and private tubs.
- **Dynamic Feedback**: Real-time card updates with a "No Results" fallback message.

### **2. Premium Mineral Profiling**
The `Amenities.html` page features a detailed technical breakdown of the water's chemical composition (Sulfur, Silica, Magnesium, etc.) presented in a high-fidelity architectural table.

### **3. Localized Experience**
Spas are situated in iconic **Navi Mumbai** landmarks such as:
- **Palm Beach Road, Vashi**
- **CBD Belapur (Parsik Hills)**
- **Kharghar (Central Park & Falls)**
- **Nerul (Wonders Park)**
- **Airoli & Seawoods**

### **4. Interactive Booking Modal**
Users can reserve private thermal sessions through an elegant modal interface that handles date selection and confirmation states.

### **5. Atmospheric UI**
- **Steam Overlay**: A dynamic, multi-layered CSS animation that mimics the mist of a hot spring.
- **Pulsing Indicators**: Real-time crowdedness status (Low, Moderate, Busy) with custom glow and pulse animations.
- **Downward Wipe Transitions**: Cinematic top-to-bottom screen wipes when navigating between pages.
